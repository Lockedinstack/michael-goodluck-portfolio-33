# Michael Goodluck Portfolio

A high-performance, minimalist personal portfolio built using an AI-first workflow.

## Tech Stack

- **Lovable** — AI-native app builder used to architect and iterate
- **Vercel** — hosting and edge deployment
- **React** — UI framework
- **Google Gemini** — AI assistance
- **Claude** — AI assistance

## Key Features

- Clean editorial design with a warm, minimalist aesthetic
- Project cards showcasing selected work
- Formspree-powered contact form
- Fully responsive layout, optimized for readability and speed

## Deployment

Deployed via Vercel.
